# Demo Script

1. Open `http://127.0.0.1:8501`.
2. Click `Run Demo` or run `make demo`.
3. Show Overview: logs processed, alert volume, threat mix, model F1.
4. Open Live Logs: filter by label, event type, source IP, or endpoint.
5. Open Threat Alerts: select a brute force or web attack alert.
6. Open LLM Advisor: generate summary, investigation steps, mitigation, and evidence checklist.
7. Open Model Metrics: explain accuracy, precision, recall, F1, classification report, and feature importance.
8. Open Rule vs AI: show agreement and disagreement cases.
9. Open Evidence: show report-ready files.
10. Close with scope: localhost, Docker network, synthetic/public data, defensive-only, advisory LLM.
