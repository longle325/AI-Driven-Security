# Demo Script

1. Open `http://127.0.0.1:8501`.
2. Click `Run Demo` or use Reset, Step, and Auto Stream to replay a SOC live scenario.
3. Show Live Log Stream: log time-series appears as network events flow through the already-trained IDS2025 model.
4. Select an abnormal event escalated into Incident Queue.
5. Show Investigation Panel: source, endpoint, rule reason, model prediction, confidence, and important features.
6. Click Generate Insight so the LLM or fallback advisor explains the incident and recommends next steps.
7. Close with scope: localhost, Docker network, synthetic/public data, defensive-only, advisory LLM, offline model training.
