# Ethics and Legal Scope

This project is a defensive, local-only security education lab.

Allowed:

- Localhost.
- Docker networks.
- Synthetic logs.
- Public IDS datasets.
- Self-built lab web app.
- Harmless simulated markers.

Not allowed:

- Scanning public IP addresses.
- Brute-forcing real systems.
- Exploiting third-party applications.
- Collecting real personal data.
- Malware, persistence, evasion, or privilege escalation guidance.
- Letting the LLM execute response actions automatically.

The LLM Advisor is advisory only. A human analyst must review all recommended actions before applying them.
